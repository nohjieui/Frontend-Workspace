        선택자 {
            display: flex;
        }
        * 요소를 통해 레이아웃을 구성시 유연하게(flex)위치를 선정할 수 있도록 도와주는 속성
        단, flex 요소끼리만 영향을 끼침.

justify-content : 요소들을 가로로 정렬하도록 도와주는 속성

        선택자 {
            justify-content: flex-start/flex-end/center/space-between/space-around/space-evenly
            flex-start : 요소들을 왼쪽으로 정렬
            flex-end : 요소들을 오른쪽으로 정렬
            center : 가운데 정렬
            space-between : 요소들 사이에 동일한 간격을 부여하여 정렬
            space-around : 요소들 주위(양옆)에 동일한 간격을 부여하여 정렬
            space-evenly : 요소들 마다 동일한 간격을 부여하여 정렬
        }

flex-direction : flex 요소들의 정렬 방향 지정(행, 열 단위)

        선택자{
            flex-diraction: row/row-reverse/column/colunm-reverse
            row : 요소들을 텍스트 방향과 동일하게 정렬
            row-reverse : 텍스트의 반대방향으로 정렬
            column : 요소들을 위에서 아래로 정렬
            colunm-reverse : 요소들을 아래에서 위로 정렬
        }

align-items : 요소들을 세로기준으로 정렬하게 도와주는 속성

        선택자 {
            align-items: flex-start/flex-end/center/stretch
            flex-start : 요소들을 맨 위로 정렬
            flex-end : 요소들을 맨 아래로 정렬
            center : 가운데로 정렬
            stretch : 요소들을 컨테이너(부모요소)의 크기에 맞도록 늘려주는 속성
        }

flex-wrap : 요소들을 한줄로 정렬할지, 여러줄로 정렬할지 정하는 속성

        선택자 {
            flex-wrap : nowrap/wrap/wrap-reverse;
            nowrap : 요소들을 한줄로만 정렬시키는 속성
            wrap : 요소들을 여러줄로 정렬시키는 속성
            wrap-reverse : 요소들을 여러줄로 반대로 정렬시키는 속성
        }
